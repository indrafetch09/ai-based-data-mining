# AI Data Miner

This project is a simple FastAPI application that demonstrates the use of Google's Gemini API to generate content. It's designed to be a starting point for building AI-powered data mining or content generation services.

## Features

-   **FastAPI Backend**: A lightweight and fast web framework for building APIs with Python.
-   **Google Gemini Integration**: Utilizes the Gemini API for advanced content generation.
-   **Environment Variable Management**: Securely handles API keys using `python-dotenv`.

## Setup

Follow these steps to set up the project locally.

### Prerequisites

-   Python 3.8+
-   `pip` (Python package installer)

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/ai-data-miner.git
    cd ai-data-miner
    ```

2.  **Create a virtual environment** (recommended):

    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

3.  **Install dependencies:**

    ```bash
    pip install -r requirements.txt
    ```

4.  **Configure environment variables:**

    Create a `.env` file in the root of the project and add your Google Gemini API key:

    ```
    GEMINI_API_KEY=your_gemini_api_key_here
    ```

    Replace `your_gemini_api_key_here` with your actual API key obtained from the Google Cloud Console or AI Studio.

## Running the Application

1.  **Activate your virtual environment** (if not already active):

    ```bash
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

2.  **Start the FastAPI application:**

    ```bash
    uvicorn main:app --reload
    ```

    The `--reload` flag will enable live-reloading of the application when code changes are detected.

3.  **Access the API:**

    Open your web browser and go to `http://127.0.0.1:8000`. You should see the response generated by the Gemini API.

    You can also access the interactive API documentation (Swagger UI) at `http://127.0.0.1:8000/docs`.
